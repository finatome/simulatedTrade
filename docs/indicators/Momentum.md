# Momentum (MOM)

| Name | Type | Prerequisite | Use Cases |
| :--- | :--- | :--- | :--- |
| Momentum (MOM) | Momentum | OHLC Data | Simplest measure of trend strength and direction. |

## Definition

The absolute difference in price over a fixed lookback period.

## Mathematical Equation

$$
C_t - C_{t-n}
$$
