# Relative Vigor Index (RVI)

| Name | Type | Prerequisite | Use Cases |
| :--- | :--- | :--- | :--- |
| Relative Vigor Index (RVI) | Momentum | EMA | Trend confirmation in volatile markets. |

## Definition

Measures the conviction of a trend by comparing close to open.

## Mathematical Equation

$$
\frac{EMA(C-O)}{EMA(H-L)}
$$
