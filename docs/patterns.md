# Candlestick Patterns

Candlestick patterns are visual representations of price movements that can signal potential future market behavior. They are formed by one or more candlesticks and are used by traders to identify trading opportunities.

## Single Candle Patterns

| Pattern | Type | Description |
| :--- | :--- | :--- |
| **[Hammer](patterns/Hammer.md)** | Bullish | Small body at top, long lower shadow. Reversal at bottom of downtrend. |
| **[Inverted Hammer](patterns/Inverted_Hammer.md)** | Bullish | Small body at bottom, long upper shadow. Reversal at bottom of downtrend. |
| **[Hanging Man](patterns/Hanging_Man.md)** | Bearish | Small body at top, long lower shadow. Reversal at top of uptrend. |
| **[Shooting Star](patterns/Shooting_Star.md)** | Bearish | Small body at bottom, long upper shadow. Reversal at top of uptrend. |
| **[Doji](patterns/Doji.md)** | Indecision | Open approx. equal to Close. Market neutrality. |
| **[Spinning Top](patterns/Spinning_Top.md)** | Indecision | Small body centered between long shadows. Consolidation. |
| **[Marubozu](patterns/Marubozu.md)** | Continuation | Long body with no shadows. Strong directional conviction. |

## Dual Candle Patterns

| Pattern | Type | Description |
| :--- | :--- | :--- |
| **[Bullish Engulfing](patterns/Bullish_Engulfing.md)** | Bullish | Large green candle engulfs previous small red candle. |
| **[Bearish Engulfing](patterns/Bearish_Engulfing.md)** | Bearish | Large red candle engulfs previous small green candle. |
| **[Piercing Line](patterns/Piercing_Line.md)** | Bullish | Green candle closes > 50% into previous red candle. |
| **[Dark Cloud Cover](patterns/Dark_Cloud_Cover.md)** | Bearish | Red candle closes > 50% into previous green candle. |
| **[Bullish Harami](patterns/Bullish_Harami.md)** | Bullish | Small green candle inside previous large red candle. |
| **[Bearish Harami](patterns/Bearish_Harami.md)** | Bearish | Small red candle inside previous large green candle. |
| **[Tweezer Bottoms](patterns/Tweezer_Bottoms.md)** | Bullish | Matching lows. Support indication. |
| **[Tweezer Tops](patterns/Tweezer_Tops.md)** | Bearish | Matching highs. Resistance indication. |
